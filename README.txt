Lancer L'api : 
npm i
npm start
Ne pas oublier d'avoir lancer WAMP ou MAMP et d'avoir crée la base de donnée grâce au script SQL fournit dans le dossier.

Lancer l'application mobile : 
npm i
ionic serve --lab --devapp
Ne pas oublier de désactiver les CORS du navigateur pour le fonctionnement de l'application en local.

Lancer le client web: 
composer i
php bin/console server:run
